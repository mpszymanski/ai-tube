Built for attention

Six things YouTube won't do for you.

AI Tube is an opinionated front-end. It reshapes search, hides the algorithm, and gives the time back to you.

Local AI rephrase

Your query goes through a local model first. It strips clickbait language, expands intent, and asks for substance — before YouTube ever sees it.

Clickbait filter

Caps, !!!, "I tried X for 30 DAYS", thumbnails screaming for attention. The filter culls them and keeps only what reads like substance.

Watch-time meter

A pill in the top bar tracks today and the week, so an hour-long hole feels like one. No streaks, no nudges — just visible truth.

Quota-aware

A live unit counter shows how much of your daily YouTube API quota you've spent. Search smarter; never get rate-limited mid-binge.

Subscriptions, calmed

A flat row of channels at the top, no infinite home feed. Pick a creator and you get their work — chronologically, end of story.

Local-first & private

No accounts, no telemetry, no third-party servers. Your API key, your model, your machine. Everything else stays where it belongs.

How it works

Four steps between you and a good video.

01 / INSTALL

Run the installer

Run the NSIS installer. No admin rights needed.

02 / API KEY

Get a YouTube API key

Free YouTube Data API v3 key from Google Cloud Console — 10,000 units/day free.

03 / FIRST LAUNCH

Download the model, once

The app downloads the Qwen3-4B model (~3 GB) on first launch. After that, everything works offline.

04 / SEARCH

Type what you want to watch

The AI figures out whether it's a topic, a channel, or both.

Install

Download. Double-click. Done.

One Windows installer. Bring your own YouTube Data API v3 key. The app downloads its model on first launch and runs offline from then on.
