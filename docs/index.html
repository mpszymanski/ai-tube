AI Tube

Privacy-first YouTube for Windows.
Local AI search, no tracking, enforced limits.

Download for Windows

Free & open source · Windows 10/11 · ~3 GB first-run model download

What it does

🤖

Local AI Search

Type in plain English. A local Qwen3-4B model understands channel names, topics, and time periods — no cloud required.

🚫

Clickbait Filter

Video titles are scored by the same local model. Sensationalist results are hidden by default; one click to reveal them.

⏱

Watch Limits

Set daily and weekly caps. The app enforces them and prevents you from raising limits once you're over.

📺

Channel Subscriptions

Follow channels and browse their latest videos — without YouTube's feed, autoplay, or recommendations.

How it works

  1. Install — run the NSIS installer, no admin rights needed.
  2. Get an API key — free YouTube Data API v3 key from Google Cloud Console (10,000 units/day free).
  3. First launch — the app downloads the Qwen3-4B model (~3 GB) once. After that, everything works offline.
  4. Search — type what you want to watch. The AI figures out whether it's a topic, a channel, or both.
